# 🎙️ Nova Voice Assistant

A modern AI-powered Voice Assistant built using **HTML, CSS, JavaScript, and Python Flask**. Nova provides voice interaction through Speech Recognition and Text-to-Speech, allowing users to execute commands, search the web, and receive spoken responses through an elegant Gemini-inspired interface.

---

## ✨ Features

- 🎤 Voice Recognition (Web Speech API)
- 🔊 Text-to-Speech Responses
- 💬 Modern Chat Interface
- 🕒 Current Time & Date
- 🔍 Google Search
- ▶️ Open YouTube
- 🌐 Open Google
- 🤖 AI-like Rule-Based Assistant
- ⚡ Flask Backend API
- 📱 Responsive User Interface
- 🎨 Gemini-inspired Dark Theme

## 🧠 How It Works

1. User clicks the microphone.
2. Browser captures voice using the Web Speech API.
3. The spoken command is sent to the Flask backend.
4. The AI Engine processes the command.
5. A structured response is returned.
6. The response is displayed in the chat interface.
7. The browser reads the response aloud using Speech Synthesis.
